> HCF (Hardcore Factions) is a Minecraft server plugin that adds a new level of difficulty and excitement to the game. It introduces a faction-based gameplay system where players can form alliances, build bases, and engage in intense PvP battles. With features like land claiming, raiding, and custom enchantments, HCF offers a unique and challenging experience for Minecraft players.

The primary goal of this project is to provide a robust and scalable foundation for HCF servers. It allows developers to easily customize and extend functionalities to meet their specific needs. With an active and collaborative community, this core is the ideal choice for launching a new server or optimizing an existing one.

### Phase #1: Core Essentials (Current)
* [ ] **Team System:** base structure, member management, and roles.
* [X] **Claim System:** Chunk-based spatial hashing for $O(1)$ lookup performance.
* [ ] **DTR (Deaths Till Raidable):** Implementation of DTR freeze, regeneration, and raid states.
* [ ] **Persistance Layer:** Integration with MongoDB for asynchronous data saving.

### Phase #2: Game Mechanichs
* [ ] **Deathban & Lives System:** Configurable deathbans based on rank and life management.
* [ ] **Economy Integration:** Vault support and internal balance handling.
* [ ] **SOTW / EOTW Modules:** Automation for "Start of the World" and "End of the World" timers and protections.
* [ ] **Combat Tag and Cooldowns:** Systems to prevent combat logging and manage PvP interactions.

### Phase #3: Events & World Control
* [ ] **KOTH (King of the Hill):** Dynamic event system with customizable rewards and scheduling.
* [ ] **Citadel & Conquest:** Territory control mechanics with capture points and faction benefits
* [ ] **World Border Management:** Dynamic world borders for different game modes and events.
* [ ] **Custom Scoreboard & Tablist:** Real-time updates with faction and player information.
* [ ] **Crowbar & Elevate Signs:** Classic HCF utility features.

### Phase #4: Quality of Life & Admin Tools
* [ ] **In-Game Commands:** Comprehensive command system for players and staff.
* [ ] **Staff Mode**: Built-in tools for moderation.
* [ ] **Detailed Logging:** Tracking block changes, diamond mining, and combat logs.
